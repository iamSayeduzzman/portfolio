# Personal Portfolio Website 🚀

A modern, responsive, and fully dynamic personal portfolio website built with Vanilla HTML, CSS, and JavaScript. The design features a beautiful "Dark space" glassmorphism theme with a smooth light/dark mode toggle.

## 📌 Important Note 

> **Feel free to copy, modify, or use this code for your own projects without any hesitation. However, in return, I would be really happy if you could drop a ⭐ on this repository and give me a follow on GitHub!** 

## ✨ Features

- **Glassmorphism UI:** Modern and sleek frosted glass effects.
- **Dark & Light Mode:** Seamlessly switch between themes.
- **Dynamic Content:** Projects, Skills, Education, and Experience data are dynamically rendered using JavaScript arrays.
- **Fully Responsive:** Perfectly optimized for Desktops, Tablets, and Mobile devices.
- **Typewriter Effect:** Animated hero section for a professional look.
- **Project Filtering:** Easily filter projects by categories (Web Dev, App Dev, etc.).
- **Interactive Modals:** Certificate viewing with a dynamic modal.

## 🛠️ Tech Stack

- **HTML5:** Semantic and accessible structure.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and custom animations.
- **JavaScript (ES6+):** DOM manipulation, Intersection Observers for scroll animations, and dynamic data rendering.
- **Icons:** Devicon & SVG icons.

## 🚀 Getting Started

To run this project locally, simply follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/iamSayeduzzman/portfolio.git](https://github.com/iamSayeduzzman/portfolio.git)

```

2. **Navigate to the directory:**
```bash
cd portfolio

```


3. **Open the project:**
Simply open the `index.html` file in your favorite web browser. No local server or build tools required!

## 📂 File Structure

* `index.html`: Main HTML file containing the structure.
* `style.css`: Contains all the styling, including dark/light mode variables and responsive media queries.
* `script.js`: Handles dynamic content rendering, typewriter effects, theme toggling, and modal interactions.
